# Domoro Restaurant Website 
NYC's Gambian Restaurant 
-------------------------------------------------
### Domoro 
- means to eat in the Mandinka language , commonly spoken in The Gambia.
  
This is a mock restaurant site based in NYC showcasing Gambian foods and culture.
